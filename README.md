📊 FPA - Federación de Pádel App
Aplicación web diseñada para la Federación de Pádel de Añatuya (FPA).
Su objetivo es ofrecer una interfaz pública para jugadores y espectadores, y un panel privado para administradores.

🚀 Funcionalidades principales
Interfaz pública (visible para todos)
Ranking: tabla de posiciones Masculino y Femenino.

Torneos: listado de torneos con fechas y opción de creación.

Inscripción: formulario simple para registrar jugadores.

Menú superior minimalista: navegación clara con efecto neón en la sección activa.

Logo FPA: centrado y protagonista en la cabecera.

Interfaz privada (solo administradores)
Dashboard lateral: gestión de jugadores, torneos e inscripciones.

Validación y control: administración de datos y configuraciones internas.
(Se implementará en fases posteriores junto al login y roles)

🛠️ Tecnologías utilizadas
React + TypeScript

React Router v6 (navegación)

TailwindCSS (estilos y paleta personalizada FPA)

@fontsource (tipografía premium: Inter/Poppins)
